# Python-for-Cybersecurity-Exercises
This repository contains a set of exercises and solutions related to the use of Python in cybersecurity.

Source: https://www.w3resource.com/python-exercises/cybersecurity/index.php

## 1. SHA-256 Hasher

Write a Python program that defines a function and takes a password string as input and returns its SHA-256 hashed representation as a hexadecimal string.

## 2. Random Password Generator

Write a Python program that defines a function to generate random passwords of a specified length. The function takes an optional parameter length, which is set to 8 by default. If no length is specified by the user, the password will have 8 characters.
